# MageOS Async Events Sinks

This is the mono repository containing all the different event sinks for MageOS
Async Events. For installation instructions please refer to the individual
package's repository.

## Available Event Sinks

### AWS

[mageos-async-events-aws](https://github.com/mage-os/mageos-async-events-aws)

- EventBridge
- SQS

### GCP

[mageos-async-events-gcp](https://github.com/mage-os/mageos-async-events-gcp)

- Pub/Sub

### Azure

[mageos-async-events-azure](https://github.com/mage-os/mageos-async-events-azure)

- Event Grid
